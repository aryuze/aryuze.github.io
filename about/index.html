· html
· C++

Programming Experience

我在小学的时候曾经找过一些编程的课程,我记得很多学生都因为老师讲的快学不会所以不学了(当时学的都是基础的python)
当时我身边的人包括父母基本都不支持我继续学编程,一个是因为浪费时间,另一个就是怕我把电脑搞坏
但是我不顾其他人的劝阻,还是坚持把python基础学完了,还自学了tkinter所有内容,后来又整了一点pygame游戏
自学的过程是非常艰苦的,基本上啥问题都需要自己解决或者上网查,很麻烦,并且常常把我逼疯.
但是学编程其实也相当于促进我学习了英语(主要是不学好英语程序报错了你压根看不懂)
我最先接触的是python,后来自学的C++,接着又学了html,js什么的.
但是当我回首过往,其实我的学习没有一个体系,并且总是断断续续(主要是因为学业繁重)
我最不擅长的还是编程的核心--也就是跟高难算法有关的部分
我会在接下来的几年里努力补上这个缺口。

My Value

我自认为我是一个思想较同龄人深刻的人，我在求学这几年真的读了很多启迪性的书籍，它们给予了我莫大的帮助
现在这个时代，发展太快了，所有人只能“边反应边适应边创造”，世界就是这样的，复杂又迅疾。
但不变的是，人们仍然为了社交而拼命，为了维持自己的复杂人际关系网而不断调整自我。
人们的所有烦恼皆来自于人际关系。
我的朋友圈子很窄，作为现代社会的一名竞争者，我显然处在不利地位。
当大家忙于享受来自不同领域的不同人对他们的讨好与照顾时，我只能孤军奋战。
起初，我认为这个世界对我是不公平的，但是后来逐渐就看开了，一切烦恼其实都没啥意义，不如享受自己独处的时光，陪陪自己。
只有你自己可以陪你自己走到最后，所以为什么不努力和自己友好相处呢?
想要在这个世界做出一点成绩，首先就要解决自己社交的问题。让自己立足。然后才是更高层次的需求和成就
社交听起来是人生很小的一部分，但是严格来说，解决了社交问题，那么人生就实现了第一层的自由。
财富自由是第二层自由，精神富足是第三层自由
想要在这个世界过的开心，就要努力实现自己的自由。
我希望我以后能够达到这种自由不羁的状态。自由的权力是自己争取到的最有难度，最有价值的东西。
